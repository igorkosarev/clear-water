# REQ-19: Список поставщиков в одну колонку

**Статус:** Done  
**Приоритет:** P2  
**Компонент:** ui  

## Описание

На странице `/suppliers` карточки поставщиков сейчас выводятся в две колонки (`grid-cols-1 sm:grid-cols-2`). На всех размерах экрана нужна одна колонка — так список читается лучше и карточки выглядят менее скученно.

## Что нужно изменить

В `src/pages/Suppliers.tsx`, строка 98:

```tsx
// было
<div className="grid grid-cols-1 sm:grid-cols-2 gap-4">

// стало
<div className="flex flex-col gap-4">
```

## Критерии приёмки

- [ ] Карточки поставщиков отображаются в одну колонку на всех размерах экрана
- [ ] Карточки занимают полную доступную ширину контейнера
- [ ] Ничего другое на странице не сломано

## Ограничения

- Не менять структуру данных и сами карточки
- Не трогать другие страницы
