# Requirement: Cross-reference modal для страниц Contaminants и Methods

## What

Заменить навигационные ссылки (`<Link to="...#id">`) на перекрёстных страницах на открытие Modal-окна с краткой карточкой связанного элемента. Пользователь остаётся на текущей странице и получает нужную информацию без потери контекста.

## Why

Сейчас клик на пилл "Boiling" (на странице Contaminants) уводит пользователя на другую страницу — он теряет место и контекст чтения. Modal позволяет получить справку о связанном методе/загрязнителе не выходя со страницы, что соответствует паттерну Wikipedia article preview и лучшим практикам энциклопедических сайтов.

## Acceptance criteria

- [ ] Клик на пилл метода (блок "Removed by" на странице Contaminants) открывает Modal с карточкой этого метода
- [ ] Клик на пилл загрязнителя (блок "Removes" на странице Methods) открывает Modal с карточкой этого загрязнителя
- [ ] Modal закрывается по клику на backdrop, по кнопке ✕ и по клавише Escape
- [ ] **Карточка метода** в Modal содержит: название, бейджи complexity + costTier, описание, блоки "How it works", "Limitations", "Typical use". Блок "Removes" (пилы загрязнителей) — отсутствует.
- [ ] **Карточка загрязнителя** в Modal содержит: название, бейдж category, описание, блоки "Sources", "Health risks", "Detection". Блок "Removed by" (пилы методов) — отсутствует.
- [ ] Иконка (emoji): присутствует, но маленькая — не доминирует в карточке (размер ≤ 32px или text-2xl)
- [ ] Анимация появления/исчезновения Modal через Framer Motion (fade + scale)
- [ ] На мобильном Modal занимает ≥ 90% ширины экрана, скроллится если контент не умещается
- [ ] `<Link>` полностью заменяются — никакой навигации на другую страницу при клике на пилл
- [ ] `npm run build` без ошибок TypeScript

## Out of scope

- Добавление CTA "Открыть полную страницу" внутри Modal (возможно в будущем, не сейчас)
- Изменение внешнего вида самих пиллов (цвет, форма)
- Анимация пиллов при наведении (остаётся как есть)
- Любые изменения вне блоков "Removed by" / "Removes"
- Страницы кроме Contaminants и Methods
